What the chapter is really about

Your textbook focuses on five big skills: recognising first-order linear equations, solving them with an integrating factor, solving second-order homogeneous equations, solving second-order non-homogeneous equations with CF + PI, and handling applications or substitutions when an equation can be reduced to a familiar form.

First order linear Integrating factor Second order homogeneous CF + PI Substitution / modelling
Exam mindset: most marks are not about memorising theory. They are about recognising the form, choosing the correct method, and keeping the algebra organised.

Three decisions that solve most of the chapter

  1. Ask what type of equation it is.
    Is it first order or second order? Is the right-hand side zero or not?
  2. Choose the method.
    First order linear \(\to\) integrating factor. Second order with RHS \(=0\) \(\to\) auxiliary equation. Second order with RHS \(\neq 0\) \(\to\) CF + PI.
  3. Use conditions last.
    Find the general solution first. Only then substitute values like \(y(1)=2\), \(I(0)=0\), or \(y'(0)=1\).

Ultra-short memory list

  • Order = highest derivative present.
  • Degree = power of the highest derivative, after the equation is algebraic in derivatives.
  • First order linear form: \(\dfrac{dy}{dx}+Py=Q\)
  • Integrating factor: \(\mathrm{IF}=e^{\int P\,dx}\)
  • Second order homogeneous: \(a\dfrac{d^2y}{dx^2}+b\dfrac{dy}{dx}+cy=0\)
  • General solution for non-homogeneous: \(y=\text{CF}+\text{PI}\)

Biggest mistakes to avoid

  • Forgetting to make the coefficient of \(\dfrac{dy}{dx}\) equal to 1 before using integrating factor.
  • Using conditions before finding the general solution.
  • Mixing up the three root cases for the auxiliary equation.
  • Choosing the wrong guessed form for the PI.
  • Dropping constants or signs when dividing by the integrating factor.

Types of questions you can expect

Exam strategy: rewrite into a standard form, identify the method, solve cleanly, then use the condition at the end. For first-order linear questions, the core memory line is \(\mu=e^{\int P\,dx}\), then \(\dfrac{d}{dx}(\mu y)=\mu Q\).

1) First order linear

Usually looks like \(\dfrac{dy}{dx}+Py=Q\) after rearranging.

\[(x+2)\frac{dy}{dx}+y=(x+2)^3 \quad \Rightarrow \quad \frac{dy}{dx}+\frac{1}{x+2}y=(x+2)^2\]

2) Second order homogeneous

Right-hand side is zero, so solve the auxiliary quadratic.

\[a\frac{d^2y}{dx^2}+b\frac{dy}{dx}+cy=0\]

3) Second order non-homogeneous

Right-hand side is not zero, so find CF first, then PI.

\[a\frac{d^2y}{dx^2}+b\frac{dy}{dx}+cy=f(x),\qquad y=\text{CF}+\text{PI}\]

4) Substitution questions

Exam may tell you to use a substitution like \(y=vx\) or \(x=e^u\) to turn the equation into a familiar type.

5) Application / modelling

Electric circuits and population growth questions test whether you can build or solve the DE and use the given conditions.

6) Solve and then find constants

Very common: “Find the general solution. Hence, find the solution given ...”

Spot the method fast

  • If it can be written as \(\dfrac{dy}{dx}+P(x)y=Q(x)\), use the integrating factor.
  • If it has \(y''\), \(y'\), and \(y\) with right-hand side 0, use the auxiliary equation.
  • If it has \(y''\), \(y'\), and \(y\) with a non-zero right-hand side, find CF first, then PI.
  • If you see a substitution such as \(y=vx\) or \(x=e^u\), the aim is to turn it into one of the standard methods above.
  • If the wording talks about a circuit, current, limiting value, or population, expect a modelling question.

Method 1 — First order linear using integrating factor

\[\frac{dy}{dx}+Py=Q\qquad \text{with} \qquad \mathrm{IF}=e^{\int P\,dx}\]
  1. Rewrite the equation so the coefficient of \(\dfrac{dy}{dx}\) is 1.
  2. Identify \(P\) and \(Q\).
  3. Find the integrating factor: \(\mathrm{IF}=e^{\int P\,dx}\).
  4. Multiply the whole equation by the IF, so the left side becomes \(\dfrac{d}{dx}(y\cdot \mathrm{IF})\).
  5. Integrate both sides.
  6. Make \(y\) the subject, then use any condition if given.
Pattern to remember: \(y\cdot \mathrm{IF}=\int Q(\mathrm{IF})\,dx + c\)

Method 2 — Second order homogeneous

\[a\frac{d^2y}{dx^2}+b\frac{dy}{dx}+cy=0\]
  1. Write the auxiliary equation: \(am^2+bm+c=0\).
  2. Solve for the roots.
  3. Choose the correct solution form from the table below.
  4. Only after that, use any conditions like \(y(0)=2\), \(y'(0)=1\).
Roots of \(am^2+bm+c=0\)General solution
Real and distinct: \(m_1, m_2\)\(y=Ae^{m_1x}+Be^{m_2x}\)
Real and equal: \(m,m\)\(y=(A+Bx)e^{mx}\)
Complex: \(\alpha \pm \beta i\)\(y=e^{\alpha x}(A\cos \beta x + B\sin \beta x)\)

Method 3 — Second order non-homogeneous

\[a\frac{d^2y}{dx^2}+b\frac{dy}{dx}+cy=f(x),\qquad y=\text{CF}+\text{PI}\]
  1. Ignore \(f(x)\) first and solve the homogeneous equation to get the CF.
  2. Guess a suitable PI based on the form of \(f(x)\).
  3. Differentiate the PI, substitute, and compare coefficients to find unknown constants.
  4. Write the full answer as CF + PI.
If \(f(x)\) looks like...Try PI of the form...
Constant \(k\)\(y_p=A\)
Linear \(mx+n\)\(y_p=Ax+B\)
Quadratic \(ax^2+bx+c\)\(y_p=Ax^2+Bx+C\)
Trig \(p\cos kx + q\sin kx\)\(y_p=A\cos kx + B\sin kx\)
Exponential \(Ce^{kx}\)\(y_p=Ae^{kx}\)
Watch out: if your guessed PI duplicates part of the CF, multiply the guess by \(x\) until it no longer clashes.

Method 4 — Substitution / reducible form

Some equations are not in the standard forms immediately, but the substitution makes them manageable.

  • For homogeneous first-order expressions involving \(x\) and \(y\), the exam may use \(y=vx\), so \(\dfrac{dy}{dx}=v+x\dfrac{dv}{dx}\).
  • For Euler-Cauchy style equations, the exam may use \(x=e^u\) so that derivatives transform into a constant-coefficient second-order equation in \(u\).
Shortcut idea: the substitution is there to turn the question into one of the earlier methods, not to create a brand-new method.

Method 5 — Modelling questions

  • Electric circuits: usually become a first-order linear DE like \(L\dfrac{dI}{dt}+RI=E(t)\).
  • Population growth: logistic model often appears as \(\dfrac{dP}{dt}=kP\left(1-\dfrac{P}{K}\right)\), which is solved by separation and partial fractions.
Key habit: read the wording carefully. They may ask for the current and the limiting current, or for the time taken to reach a population size.

Practice questions

Try each one first. Then open the hint, steps, and answer.

1 First order linear — integrating factor

Solve \(\dfrac{dy}{dx}+2y=x^2\).

Hint
It is already in the form \(\dfrac{dy}{dx}+Py=Q\). Here \(P=2\), so the integrating factor is \(e^{\int 2\,dx}=e^{2x}\).
Steps
  1. \(\mathrm{IF}=e^{2x}\)
  2. Multiply through: \(e^{2x}\dfrac{dy}{dx}+2e^{2x}y=x^2e^{2x}\)
  3. So \(\dfrac{d}{dx}(ye^{2x})=x^2e^{2x}\)
  4. Integrate: \(ye^{2x}=\int x^2e^{2x}\,dx + c\)
  5. Using integration by parts twice, \(\int x^2e^{2x}\,dx=e^{2x}\left(\dfrac{x^2}{2}-\dfrac{x}{2}+\dfrac14\right)\)
  6. Therefore \(y=\dfrac{x^2}{2}-\dfrac{x}{2}+\dfrac14+ce^{-2x}\)
Answer
\[y=\frac{x^2}{2}-\frac{x}{2}+\frac14+ce^{-2x}\]
2 First order linear with condition

Solve \(x\dfrac{dy}{dx}+y=x^3\), given that \(y=2\) when \(x=1\).

Hint
First divide by \(x\), then use integrating factor. Use the condition only after the general solution.
Steps
  1. Rewrite as \(\dfrac{dy}{dx}+\dfrac1x y=x^2\)
  2. \(\mathrm{IF}=e^{\int \frac1x\,dx}=e^{\ln x}=x\)
  3. Multiply through: \(\dfrac{d}{dx}(xy)=x^3\)
  4. Integrate: \(xy=\dfrac{x^4}{4}+c\)
  5. So \(y=\dfrac14x^3+\dfrac{c}{x}\)
  6. Use \(y(1)=2\): \(2=\dfrac14+c\Rightarrow c=\dfrac74\)
Answer
\[y=\frac14x^3+\frac{7}{4x}\]
3 Second order homogeneous — distinct roots

Solve \(\dfrac{d^2y}{dx^2}-5\dfrac{dy}{dx}+6y=0\).

Hint
Write the auxiliary equation and factorise it.
Steps
  1. Auxiliary equation: \(m^2-5m+6=0\)
  2. Factorise: \((m-2)(m-3)=0\)
  3. Roots are \(m=2,3\), real and distinct
  4. So the general solution is \(y=Ae^{2x}+Be^{3x}\)
Answer
\[y=Ae^{2x}+Be^{3x}\]
4 Second order homogeneous — repeated root

Solve \(\dfrac{d^2y}{dx^2}-4\dfrac{dy}{dx}+4y=0\).

Hint
If the auxiliary quadratic has one repeated root \(m\), use \((A+Bx)e^{mx}\).
Steps
  1. Auxiliary equation: \(m^2-4m+4=0\)
  2. \((m-2)^2=0\), so the repeated root is \(m=2\)
  3. Therefore \(y=(A+Bx)e^{2x}\)
Answer
\[y=(A+Bx)e^{2x}\]
5 Second order homogeneous — complex roots

Solve \(\dfrac{d^2y}{dx^2}+2\dfrac{dy}{dx}+5y=0\).

Hint
For roots \(\alpha\pm\beta i\), use \(e^{\alpha x}(A\cos\beta x+B\sin\beta x)\).
Steps
  1. Auxiliary equation: \(m^2+2m+5=0\)
  2. \(m=\dfrac{-2\pm\sqrt{4-20}}{2}=-1\pm 2i\)
  3. So \(\alpha=-1\) and \(\beta=2\)
  4. Hence \(y=e^{-x}(A\cos 2x+B\sin 2x)\)
Answer
\[y=e^{-x}(A\cos 2x+B\sin 2x)\]
6 Second order non-homogeneous — polynomial PI

Solve \(y''-3y'+2y=4x+1\).

Hint
Find the CF from the homogeneous equation. For the PI, try \(y_p=Ax+B\).
Steps
  1. Homogeneous part: \(m^2-3m+2=0\Rightarrow (m-1)(m-2)=0\)
  2. So \(\text{CF}=C_1e^x+C_2e^{2x}\)
  3. Try \(y_p=Ax+B\), then \(y_p'=A\), \(y_p''=0\)
  4. Substitute: \(0-3A+2(Ax+B)=4x+1\)
  5. Compare coefficients: \(2A=4\Rightarrow A=2\)
  6. Constant terms: \(-3A+2B=1\Rightarrow -6+2B=1\Rightarrow B=\tfrac72\)
Answer
\[y=C_1e^x+C_2e^{2x}+2x+\frac72\]
7 Second order non-homogeneous — exponential PI

Solve \(y''-y'-2y=3e^{x}\).

Hint
The RHS is exponential, so try \(y_p=Ae^x\). Check first that \(e^x\) is not already part of the CF.
Steps
  1. Homogeneous part: \(m^2-m-2=0\Rightarrow (m-2)(m+1)=0\)
  2. \(\text{CF}=C_1e^{2x}+C_2e^{-x}\)
  3. Try \(y_p=Ae^x\), so \(y_p'=Ae^x\), \(y_p''=Ae^x\)
  4. Substitute: \(Ae^x-Ae^x-2Ae^x=3e^x\)
  5. So \(-2A=3\Rightarrow A=-\tfrac32\)
Answer
\[y=C_1e^{2x}+C_2e^{-x}-\frac32e^x\]
8 Modelling — circuit current

Given \(2\dfrac{dI}{dt}+6I=30\) and \(I(0)=0\), find \(I\) in terms of \(t\).

Hint
First divide by 2. This becomes a first-order linear equation in \(I\).
Steps
  1. Rewrite: \(\dfrac{dI}{dt}+3I=15\)
  2. Integrating factor: \(e^{\int 3dt}=e^{3t}\)
  3. Then \(\dfrac{d}{dt}(Ie^{3t})=15e^{3t}\)
  4. Integrate: \(Ie^{3t}=5e^{3t}+c\)
  5. So \(I=5+ce^{-3t}\)
  6. Use \(I(0)=0\): \(0=5+c\Rightarrow c=-5\)
Answer
\[I=5-5e^{-3t}\]

Past paper style questions

These are adapted from the past-paper style questions on your Pure Math Unit 2 Module 3 revision page.

P1 Past paper style — product rule shortcut

Solve the initial value problem \(x^2\dfrac{dy}{dx}+2xy=\cos x\), given that \(y(\pi)=0\).

Hint
Recognise the left side as \(\dfrac{d}{dx}(x^2y)\).
Steps
  1. Use the product rule in reverse: \(x^2\dfrac{dy}{dx}+2xy=\dfrac{d}{dx}(x^2y)\).
  2. So \(\dfrac{d}{dx}(x^2y)=\cos x\).
  3. Integrate: \(x^2y=\sin x + C\).
  4. Apply \(y(\pi)=0\): \(0=\sin\pi + C\Rightarrow C=0\).
  5. Hence \(y=\dfrac{\sin x}{x^2}\).
Answer
\[y= rac{\sin x}{x^2}\]
P2 Past paper style — trig first-order equation

A differential equation is given as \(\dfrac{dy}{dx} an x+y-2\cos^2x=0\).

Hint
Divide through by \( an x\) and use trig identities. This becomes linear in \(y\).
Steps
  1. Rewrite as \(\dfrac{dy}{dx}+y\cot x=2\cos^2x\cot x\).
  2. The integrating factor is \(e^{\int \cot x\,dx}=e^{\ln(\sin x)}=\sin x\).
  3. Multiply through: \(\dfrac{d}{dx}(y\sin x)=2\cos^3x\).
  4. Integrate: \(y\sin x=2\int \cos^3x\,dx\).
  5. Use \(\cos^3x=\cos x(1-\sin^2x)\) to integrate.
  6. Simplify to get the general solution below.
Answer
\[y= rac{5}{3}+ rac{1}{3}\cos 2x + C\csc x\]
P3 Past paper style — show that, then use the condition

A differential equation is given as \(x\dfrac{dy}{dx}+y=2\sin x\).

Hint
The left side is \(\dfrac{d}{dx}(xy)\).
Steps
  1. Recognise \(x\dfrac{dy}{dx}+y=\dfrac{d}{dx}(xy)\).
  2. So \(\dfrac{d}{dx}(xy)=2\sin x\).
  3. Integrate: \(xy=-2\cos x + C\).
  4. Divide by \(x\): \(y=\dfrac{C}{x}-\dfrac{2\cos x}{x}\).
  5. If \(y=2\) when \(x=\pi\), then \(2=\dfrac{C}{\pi}+\dfrac{2}{\pi}\).
  6. So \(C=2\pi-2\), giving the particular solution below.
Answer
\[y= rac{C}{x}- rac{2\cos x}{x}\quad ext{and if }y(\pi)=2,\; y= rac{2\pi-2}{x}- rac{2\cos x}{x}\]
P4 Past paper style — exponential forcing with modification

Solve \(y''-y'-2y=3e^{2x}\), given that \(y(0)=0\) and \(y'(0)=7\).

Hint
Because \(e^{2x}\) already appears in the CF, the PI must be multiplied by \(x\).
Steps
  1. Auxiliary equation: \(m^2-m-2=0\Rightarrow (m-2)(m+1)=0\).
  2. So \(y_c=Ae^{2x}+Be^{-x}\).
  3. Since the RHS is \(3e^{2x}\) and \(e^{2x}\) is already in the CF, try \(y_p=\lambda xe^{2x}\).
  4. Substitute and solve to get \(\lambda=1\), so \(y_p=xe^{2x}\).
  5. Hence \(y=Ae^{2x}+Be^{-x}+xe^{2x}\).
  6. Use \(y(0)=0\) and \(y'(0)=7\) to get \(A=2\) and \(B=-2\).
Answer
\[y=2e^{2x}-2e^{-x}+xe^{2x}\]
